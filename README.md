# riverpod_paging_sample

- Riverpodを使って汎用的なページング実装を行うサンプルです。
- AsyncNotifierを継承したクラスを用意して、手軽にページングの実装が行えるようにします
- 複数のページング形式に対応しています。
  - Page Based Paging
  - Offset Based Paging
  - Cursor Based Paging
